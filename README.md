# Adam Asmaca Oyunu (Java Swing)

**Java Swing kullanılarak geliştirilmiş masaüstü kelime tahmin oyunudur.**
**Oyunda şifre sistemi, loglama ve süre takibi bulunmaktadır.**

---

# Özellikler
- Kelime tahmin oyunu
- Swing GUI arayüzü
- Maven tabanlı proje yapısı

---

# 🛠️ Kullanılan Teknolojiler
- Java
- Swing (GUI)
- Maven
- NetBeans IDE

---

# Çalıştırma
- Projeyi NetBeans ile açın
- Maven dependency’lerini yükleyin
- P2Oyun klasörünü **Yerel Disk (C:) > P2Oyun** olacak şekilde ayarlayın.
